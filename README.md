# acr-uncrustify

Extensao que verifica a identacao de arquivos usando a ferramenta uncrustify

1. Propriedade config se refere ao caminho para o arquivo .cfg do uncrustify

Arquivo config.json

```json
{
  "config": ""
}
```

Dependencias

- uncrustify

```shell
sudo apt-get install uncrustify
```
